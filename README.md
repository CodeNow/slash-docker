## Build and Run

### Requirements

- Ruby (v2 or above)
- RubyGems
- Bundler (bundler.io)

### Development

1. Clone this repo.
2. Run `bundle update`
3. Run `bundle install`
4. Run `bundle exec jekyll build`
5. Run `bundle exec jekyll serve`
6. Open `localhost:4000/docker/` in your browser.
