# Slash Docker

## Requirements
- Linux, Unix, or OS X
- Ruby (v2 or above)
- RubyGems
- Bundler (bundler.io)

## Development
0. Make sure all requirements are met.
1. `bundle update`
2. `bundle install`
3. `bundle exec jekyll build`
4. `bundle exec jekyll serve`
5. Open `localhost:4000/docker/` in your browser.
